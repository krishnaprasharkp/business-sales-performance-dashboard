# Business Sales Performance Dashboard

This project analyzes sales performance using Power BI.

## Dashboard Features
• Total Revenue KPI
• Total Orders KPI
• Total Customers KPI
• Revenue by Region
• Revenue Trend Over Time
• Revenue by Category
• Top Performing Products

## Tools Used
Power BI
Data Visualization
Business Analytics

## Dataset
Sample retail sales dataset including:
Orders
Customers
Products
Regions

## Insights
• West region generates the highest revenue
• Technology category dominates sales
• Revenue shows strong growth trend after 2016

## Preview
Dashboard created using Power BI to help businesses monitor sales performance and identify growth opportunities.
